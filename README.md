# Spam Email Classifier

## Overview
This project uses Machine Learning to classify emails/messages as Spam or Not Spam.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Naive Bayes

## Dataset
SMS Spam Collection Dataset

## Accuracy
Approximately 97%–99%

## How to Run

1. Install dependencies
   pip install -r requirements.txt

2. Run the project
   python spam_classifier.py

3. Enter email text to test predictions.
